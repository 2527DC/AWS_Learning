# EC2 Instance Families

| Family                    | Purpose                   | Example Types    |
| ------------------------- | ------------------------- | ---------------- |
| **General Purpose**       | Balanced CPU + memory     | t3, t4g, m5, m6g |
| **Compute Optimized**     | High CPU, less memory     | c5, c6g, c6i     |
| **Memory Optimized**      | High memory, moderate CPU | r5, r6g, x1e     |
| **Storage Optimized**     | High I/O, fast storage    | i3, d3, h1       |
| **Accelerated Computing** | GPU / FPGA workloads      | p3, g4, f1       |
